OMG IT WORKS!

Congratulations, you didn't screw anything up...yet

You can make changes to the sass, javascript, and images by heading on over to the ./src folder. You can modify this page directly in ./httpdocs/index.html. Never manually add or remove files from ./httpdocs/assets. The CSS, JS, images, etc, in there are all automatically generated for you from ./src, so they'll just get overwritten.

If you're going to be using a CMS, here's the deal:

ExpressionEngine

By default, the server will watch ./src/templates. Just set the path to the templates folder in your EE config file:

  1. Create a (or move your existing) templates folder in ./src
  2. Open ./system/expressionengine/config/config.php and add (or modify) this line before "// END EE config items" around line 20-30:
    $config['tmpl_file_basepath'] = realpath('../src/templates/') . '/';
  3. Run gulp to launch the server. Profit.

Your directory structure should look like this if you installed EE correctly:

  • config.js
  • gruntfile.js
  • httpdocs
    • .htaccess
    • admin.php
    • assets
    • images
    • index.php
    • themes
  • node_modules
  • package.json
  • README.md
  • src
    • images
    • js
    • sass
    • templates
      • default_site
        • ...
          • EE serves html/php straight from here
  • system

WordPress

If you're using WordPress, I'm sorry. If you haven't quit your job already, here's how to set it up:

  1. Create a new theme folder (if it doesn't already exist) such as yourawesometheme in ./httpdocs/wp-content/themes
  2. Open ./config.js and update:
    line 25 to: config.web.assets = config.web.root + '/wp-content/themes/yourawesometheme';
    line 30 to: config.web.templates = 'config.web.assets + '/';
  3. Delete your ./httpdocs/assets folder and rerun gulp setup
  4. Run gulp to launch the server. Profit.

Your directory structure should look like this if you installed WP correctly:

  • config.js
  • gruntfile.js
  • httpdocs
    • .htaccess
    • index.php
    • ...
    • wp-admin
    • wp-content
      • themes
        • yourawesometheme
          • css
          • fonts
          • images
          • js
          • any folders you made in ./src/templates
          • any php/html files from ./src/templates
    • ...
  • node_modules
  • package.json
  • README.md
  • src
    • templates
      • all folders and php/html files placed here get synced to yourawesometheme to be served by WP
    • images
    • js
    • sass
